2. Description of Service

InkVoice is a local-first reading application with on-device Text-to-Speech (TTS).

  • TXT / EPUB reading
  • Local file import
  • Wi-Fi transfer (local network only)
  • On-device TTS (no cloud processing)
  • Advertisements (free version)
  • Membership subscription (premium features)

3. Data Collection

All data (books, audio, notes, TTS) is processed and stored locally on your device.

All data is stored locally. Uninstalling the app removes all data.
